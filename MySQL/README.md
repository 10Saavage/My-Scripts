# MySQL

## Permissions

``` bash
chmod +x mysql
```
## Usage

### Start the MySQL service

use parameter **start**
``` bash
./mysql start
```

you can also make this
``` bash
./mysql
```


  
### Stop the MySQL Service

use parameter **stop**
``` bash
./mysql stop
```

### Chech the status of MySQL

use parameter **status**
``` bash
./mysql status
```
### Log on the Command line

use parameter **connect**
``` bash
./mysql connect
```

### Display Help

use parameter **-h**

``` bash
./mysql -h
```
