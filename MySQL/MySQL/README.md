# MySQL

## Permissions

``` bash
chmod +x manoel_mysql
```
## Usage

### Start the MySQL service

use parameter **start**
``` bash
./manoel_mysql start
```

you can also make this
``` bash
./manoel_mysql
```


  
### Stop the MySQL Service

use parameter **stop**
``` bash
./manoel_mysql stop
```

### Chech the status of MySQL

use parameter **status**
``` bash
./manoel_mysql status
```
### Log on the Command line

use parameter **connect**
``` bash
./manoel_mysql connect
```

### Display Help

use parameter **-h**

``` bash
./manoel_mysql -h
```
