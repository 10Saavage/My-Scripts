# Elasticsearch

## Permissions

```bash
chmod +x elasticsearch
```

## Usage

### Start elasticsearch service
use parameter **start**

```bash
./elasticsearch  start
```

### Stop elasticsearch service
use parameter **stop**

```bash
./elasticsearch stop
```

### check the status of elasticsearch service
use parameter **status**

```bash
./elasticsearch status
```

### Display help
use parameter **help**

```bash
./elasticsearch help
```
